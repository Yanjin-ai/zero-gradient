# 幻灯提纲 —— ZeroBP-4B 边界研究 + 可训练注意力对照(11 页)

演讲时长约 12–15 分钟。关键视觉素材:能力雷达(`runs/track1_radar.png`)、能力对比图(`figures/capability_comparison.png`)、多步之墙(`figures/reasoning_depth_wall.png`)、记分卡(`figures/scorecard.png`)。数字可追溯至 `EXPERIMENT_LEDGER.md`。

---

**1 — 标题 / 一句话论点**
- "是算法,还是架构?" —— 一个无反向传播的 4B 语言模型、它的能力边界,以及一个解释这一切的可训练注意力对照实验。
- 姓名、单位、仓库/报告指针。

**2 — 动机(两个问题)**
- 在**完全无全局反向传播**、**单 T4 / 3 小时**约束下,能保住多少语言能力?
- 当它在某任务上失败:是**算法**(梯度不够)、**架构**(骨架无法表示)、还是**任务本身**?通常无法区分 —— 本研究把它拆开。
- 方法预告:**三条线**(提交 → 边界 → 对照)。

**3 — 提交线:ZeroBP-4B**
- 41.6 亿内容路由 MoE;**零 autograd**,每次更新是局部规则;确定性稀疏预算(每步 0.42% 专家);冻结注意力 reservoir + 末位塌缩。
- 结果:WikiText-103 困惑度 **≈1391 / ≈1355**,比 unigram 好约 51%;能装进 T4(~8.3GB,~8546 tok/s);确定性、有 gate、纯净性校验。→ 一个干净的研究对象。

**4 — 边界线:任务 × 方法矩阵** *(表)*
- 情感(bag)/ NLI(关系)/ 两步算术(多步),公平读出。
- 情感 **60→79%**(embedding BP 是杠杆);NLI 4B 上 **chance→chance**;算术任何 BP 深度 **chance**。
- 要点:bag 结构能装进;关系与多步不行。

**5 — "关系结构在哪里?"(三个 v2.0 探针)**
- 不塌缩读出 → **掉到 chance**(结构不在表示任何位置)。
- 可训练 attention 单独 → **+0.0pp**(权重确有更新;embedding 冻结)。
- 更深 BP → 只在小配置 65.7%,**不转移 4B**;算术任何深度都死。
- → 边界是**结构性的**;两个假设在此被推翻(诚实标注)。

**6 — 这逼出的问题**
- "ZeroBP 做不了 NLI",是 ZeroBP 的问题、BP 预算的问题、还是**骨架**的问题(冻结 reservoir + 末位塌缩)?
- 引入**对照**:只换骨架,其余保持标准。

**7 — 架构对照(第五阶段):设计 + 首验**
- 标准可训练注意力 Transformer,mean-pool,全 BP;与提交线严格隔离。
- **G0(同样的合成任务):** NLI **100%**、两步算术 **100%**,仅 **0.8M** 参数。
- → 边界是**架构性的,不是任务不可学**。

**8 — 对照骨架在真实数据上(SNLI + 多步)** *(记分卡开始)*
- **真实 SNLI:69.97%**(12.4M)vs ZeroBP chance 33.4% —— 关系结构可安装。
- **多步深度:** k≤3 = 100%;**k≥4 是墙**,从 4.7M/6k → 21.3M/15k 不动 → **不是**欠容量。

**9 — 诚实的上限(G2b + SST-2)+ 读出教训**
- 真实 **GSM8K:2%** exact-match(小 char-LM stretch,提前标注)。
- 真实 **SST-2 ≈ chance** —— 在修正一个闭式头**坍缩**(平坦 49% 是假象;BP 探针 → 53%)之后。
- 教训:用**公平、任务无关的探针**测能力(本项目被咬两次)。

**10 — 最终记分卡 + 能力雷达** *(记分卡表 + `runs/track1_radar.png`)*
- 一页:5 行 对照 vs ZeroBP 表 + 雷达图。
- 一句话故事:**新骨架跨过了 ZeroBP 的关系/浅多步边界;两栈都有诚实上限。**

**11 — 结论与未来工作**
- ZeroBP-4B:资源受限下的无反向传播基线 + 边界测量工具(擅长 LM/bag,关系/多步硬限)。
- 对照骨架:证明 NLI/浅多步*可学*,提出多步*深度*这一新研究问题。
- 下一步:用 curriculum / chain-of-thought(而非纯扩规模)打破 k≥4 墙;在可训练底座上验证更先进的 ZeroBP 规则;把对照骨架在关系/推理维度上规模化。
